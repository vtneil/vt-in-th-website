Detailed Description

template<typename T, size_t Row, size_t Col>
class vt::impl::numeric_matrix_static_t< T, Row, Col >

Numeric matrix template class where the dimension must be known at compile-time and can't be changed by any ways during runtime to prevent unexpected behavior or errors. This avoid heap usages for safety-critical and performance-critical systems.
